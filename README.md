# Capybara Jingle Maker

A cute music creator playground app inspired by Animal Crossing New Leaf's town jingle maker!

## Features

- **Note selection** with colorful buttons (A, B, C, D, E, F, G, and rest "-")
- **Bell-like sound generation** using Web Audio API for cute, light bell tones
- **Play functionality** that plays your composition sequentially
- **Visual feedback** with bouncing animations when notes play

## How to Use

1. Open `index.html` in a web browser
2. Select a note from the note selector buttons at the bottom
3. Click on any slot in the grid to place that note
4. Click "Play" or "Done" to hear your composition
5. Click "Reset" to clear the grid and start over

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- Uses Web Audio API for sound generation
- Responsive design that works on desktop and mobile
- Bell sounds created with harmonic synthesis for a rich, cute tone

## Browser Compatibility

Works best in modern browsers that support Web Audio API (Chrome, Firefox, Safari, Edge).
